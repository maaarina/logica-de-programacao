Outros loops

"Do While"

var numero = prompt("Digite um número");
numero = parseInt(numero);

var indice = 11;
do {
document.write(numero + " x " + indice + " = " + (numero * indice) + " <br> ");

indice = indice + 1;
} while (indice <= 10); document.write ("fim do loop")

O "Do While" irá fazer com que a ação seja executada ao menos uma vez, independete do valor da variável 'indice'. Neste caso ela será executada apenas uma vez, mas, se var indice = 1 e while (indice <= 10) então a ação seria executa dez vezes
